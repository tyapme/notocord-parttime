# ショートカット一覧

## 概要

notocord で使用できるキーボードショートカットの一覧です。

## 全般

| ショートカット | 動作 |
|---------------|------|
| `Esc` | モーダル/ダイアログを閉じる |
| `Enter` | フォームを送信 |
| `Tab` | 次の入力フィールドへ移動 |
| `Shift + Tab` | 前の入力フィールドへ移動 |

## フォーム

| ショートカット | 動作 |
|---------------|------|
| `Enter` | フォーム送信（ボタンフォーカス時） |
| `Space` | チェックボックス/ラジオボタンの切替 |
| `↑ / ↓` | セレクトボックスの選択変更 |

## ナビゲーション

| ショートカット | 動作 |
|---------------|------|
| `Tab` | フォーカスを次の要素へ |
| `Shift + Tab` | フォーカスを前の要素へ |
| `Enter / Space` | ボタン/リンクの実行 |

## モーダル

| ショートカット | 動作 |
|---------------|------|
| `Esc` | モーダルを閉じる |
| `Tab` | モーダル内でフォーカス移動 |

## カレンダー

| ショートカット | 動作 |
|---------------|------|
| `← / →` | 日付を前後に移動 |
| `↑ / ↓` | 週を前後に移動 |
| `Enter` | 日付を選択 |
| `Esc` | カレンダーを閉じる |

## アクセシビリティ

notocord は、キーボードのみでの操作に対応しています。

### フォーカスインジケーター

フォーカスされている要素は、視覚的にハイライトされます。

### スクリーンリーダー

主要な要素には適切な ARIA 属性が設定されています。

## ブラウザショートカット

| ショートカット | 動作 |
|---------------|------|
| `Ctrl/Cmd + R` | ページを再読み込み |
| `Ctrl/Cmd + F` | ページ内検索 |
| `F12` | 開発者ツールを開く |
