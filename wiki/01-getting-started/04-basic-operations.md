# 基本操作

## 画面構成

### ナビゲーション

notocord の画面は、下部のタブナビゲーションで構成されています。

#### スタッフの場合
| タブ | 画面 | 説明 |
|------|------|------|
| ホーム | `/home` | メイン画面 |
| 申請 | `/new` | 新規シフト申請 |
| 履歴 | `/my` | 自分の申請一覧 |

#### レビュワーの場合
| タブ | 画面 | 説明 |
|------|------|------|
| ホーム | `/home` | メイン画面 |
| 承認 | `/review` | 申請の承認管理 |
| 代理 | `/shift/proxy` | 代理申請 |
| スタッフ | `/users` | スタッフ別表示 |

#### 管理者の場合
| タブ | 画面 | 説明 |
|------|------|------|
| ホーム | `/home` | メイン画面 |
| 承認 | `/review` | 申請の承認管理 |
| 代理 | `/shift/proxy` | 代理申請 |
| スタッフ | `/users` | スタッフ別表示 |
| 管理 | `/admin` | ユーザー管理 |

## スタッフの基本操作

### 新規シフト申請

1. **「申請」タブをタップ**
2. **申請タイプに応じた入力**

   #### Fix タイプ（固定シフト）
   - 日付を選択
   - 開始時刻を入力
   - 終了時刻を入力
   - メッセージ（任意）を入力
   - 「申請」ボタンをタップ

   #### Flex タイプ（フレックス）
   - 対象週を選択
   - 希望時間数を入力（1〜40）
   - メッセージ（任意）を入力
   - 「申請」ボタンをタップ

### 申請の確認

1. **「履歴」タブをタップ**
2. **申請一覧を確認**
   - 「すべて」: 全申請を表示
   - 「確定」: 承認済みのみ表示
3. **申請をタップして詳細表示**

### 申請の編集

※ ステータスが「保留中」の申請のみ編集可能

1. **編集したい申請をタップ**
2. **「編集」ボタンをタップ**
3. **内容を変更**
4. **「保存」ボタンをタップ**

### 申請の取り下げ

1. **取り下げたい申請をタップ**
2. **「取り下げ」ボタンをタップ**
3. **取り下げ理由を入力**
4. **確認して実行**

## レビュワーの基本操作

### 承認待ち申請の確認

1. **「承認」タブをタップ**
2. **「承認待ち」タブを選択**
3. **申請一覧を確認**

### 申請の承認

1. **承認したい申請をタップ**
2. **承認モーダルが表示**
3. **「承認」を選択**
4. **メッセージ（任意）を入力**
5. **「確定」ボタンをタップ**

### 変更承認

申請内容を調整して承認する場合：

1. **申請をタップ**
2. **「変更承認」を選択**
3. **内容を調整**
   - Fix: 時間を変更
   - Flex: 時間数を変更
4. **変更理由を入力（Fix の場合必須）**
5. **「確定」ボタンをタップ**

### 申請の却下

1. **申請をタップ**
2. **「却下」を選択**
3. **メッセージ（任意）を入力**
4. **「確定」ボタンをタップ**

### 代理申請

1. **「代理」タブをタップ**
2. **対象スタッフを選択**
3. **申請内容を入力**
4. **「作成」ボタンをタップ**

※ 代理申請は即時承認される

## 管理者の基本操作

### ユーザーの追加

1. **「管理」タブをタップ**
2. **「ユーザー追加」ボタンをタップ**
3. **ユーザー情報を入力**
   - 名前
   - メールアドレス
   - ロール（staff/reviewer/admin）
   - 申請タイプ（fix/flex）
4. **「作成」ボタンをタップ**

### ユーザーの編集

1. **編集したいユーザーをタップ**
2. **「編集」ボタンをタップ**
3. **情報を変更**
4. **「保存」ボタンをタップ**

### ユーザーの有効/無効

1. **対象ユーザーをタップ**
2. **「有効/無効」トグルを変更**
3. **確認して実行**

## 共通操作

### 履歴の確認

各申請の詳細画面で：

1. **「変更履歴を見る」をタップ**
2. **履歴一覧が展開**
3. **各変更の詳細を確認**

### 検索・フィルター

承認画面（レビュワー・管理者）：

1. **「すべて」タブを選択**
2. **検索ボックスにスタッフ名を入力**
3. **該当する申請が表示**

### 過去の申請表示

1. **「過去を表示」トグルをオン**
2. **過去の申請が表示**

## キーボードショートカット

|---------------|------|

| `Enter` | フォーム送信 |
