# システム要件

## 動作環境

### サポートブラウザ

| ブラウザ | 最低バージョン | 推奨バージョン |
|----------|--------------|---------------|
| Google Chrome | 100+ | 最新版 |
| Firefox | 100+ | 最新版 |
| Safari | 15+ | 最新版 |
| Microsoft Edge | 100+ | 最新版 |

### モバイル対応

| プラットフォーム | 対応状況 |
|-----------------|---------|
| iOS Safari | ✅ 対応 |
| Android Chrome | ✅ 対応 |
| PWA | ✅ 対応 |

## ハードウェア要件

### クライアント（ユーザー）

| 項目 | 最小要件 | 推奨 |
|------|---------|------|
| 画面サイズ | 320px 以上 | 768px 以上 |
| メモリ | 2GB | 4GB 以上 |
| ネットワーク | 3G | 4G/Wi-Fi |

### 開発環境

| 項目 | 最小要件 | 推奨 |
|------|---------|------|
| CPU | 2コア | 4コア以上 |
| メモリ | 4GB | 8GB 以上 |
| ストレージ | 10GB | 20GB 以上 |

## ソフトウェア要件

### 開発環境

| ソフトウェア | 必須バージョン |
|-------------|---------------|
| Node.js | 20.x 以上 |
| pnpm | 8.x 以上 |
| Git | 2.x 以上 |

### サービス

| サービス | 用途 |
|---------|------|
| Supabase | データベース・認証 |
| Vercel | ホスティング（推奨） |

## ネットワーク要件

### 必要なポート

| ポート | 用途 |
|--------|------|
| 443 (HTTPS) | 本番環境通信 |
| 3000 (HTTP) | ローカル開発 |

### 必要なドメインアクセス

| ドメイン | 用途 |
|---------|------|
| *.supabase.co | Supabase API |
| *.vercel.app | Vercel ホスティング |
| fonts.googleapis.com | Google Fonts |

## PWA 要件

### インストール条件
- HTTPS 接続
- Service Worker 対応ブラウザ
- マニフェストファイルの存在

### オフライン機能
- 現在のバージョンではオフライン機能は限定的
- 認証にはネットワーク接続が必要

## アクセシビリティ

### 対応基準
- キーボードナビゲーション対応
- スクリーンリーダー基本対応
- ハイコントラストモード対応（準備中）

## タイムゾーン

- システムは JST (日本標準時) を基準として動作
- サーバー側では UTC で保存、クライアント側で JST 表示
